# xpans Violet Audio Encoder
Adds an audio encoder output to xpans Violet

[![Crates.io Version](https://img.shields.io/crates/v/violet_audio_encoder)](https://crates.io/crates/violet_audio_encoder)
[![docs.rs](https://img.shields.io/docsrs/violet_audio_encoder)](https://docs.rs/violet_audio_encoder/0.1.0/violet_audio_encoder/)

## Supported Formats
The Violet Audio Encoder only supports WAV encoding. Support for other formats
is planned.
